# Product Overview

The Mandelbrot Visualizer is a high-performance web application that renders the Mandelbrot set fractal with real-time pan and zoom interactions.

## Purpose

Provide an interactive, educational tool for exploring the Mandelbrot set while demonstrating the performance capabilities of WebAssembly across multiple programming languages (Rust, C++, Go).

## Key Features

- **Full-screen Canvas Rendering**: Immersive fractal visualization that fills the browser viewport
- **Interactive Pan & Zoom**: Click-and-drag panning and mouse wheel zooming for exploration
- **Multiple WebAssembly Implementations**: Compare performance across Rust, C++, and Go
- **Real-time Module Switching**: Hot-swap between different Wasm modules while preserving viewport
- **Smooth Color Gradients**: Beautiful color mapping to visualize iteration depths
- **Responsive Design**: Adapts to window resizing automatically
